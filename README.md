# My Work Salary

En mobilapp för skiftarbetare att registrera arbetspass, hålla koll på arbetstid och kontrollera att lönen från arbetsgivaren stämmer.

## 🎯 Målgrupp
- **Vårdpersonal** (undersköterskor, sjuksköterskor)
- **Alla skiftarbetare** som behöver kolla sin lön

## ✅ Huvudfunktioner

### Grundläggande
- **Jobbhantering** - Skapa och växla mellan flera jobb
- **Passregistrering** - Arbetspass, sjukdagar, semester
- **Flex-timbalans** - Spårning av över-/undertid
- **Dashboard** - Månadsstatistik och översikt
- **Svensk formatering** - Kronor, datum och tider

### Pågående utveckling
- **OB-tillägg** - Automatisk beräkning
- **Svensk helgkalender** - Helgdagsdetektering

### Framtida funktioner
- **VAB-registrering** - Vård av barn
- **Export** - Rapporter till PDF/Excel
- **Lönejämförelse** - Beräknad vs faktisk lön

## 🛠 Teknik
- **.NET MAUI** - Cross-platform mobilapp
- **SQLite** - Lokal databas
- **MVVM** - Modern arkitektur
- **Plattformar** - Android (primärt), iOS

## 📱 Användning
1. Skapa din jobbprofil
2. Registrera arbetspass
3. Se löneberäkning och flex-saldo
4. Få kontroll över din arbetssituation

## 🚀 Snabbstart
```bash
git clone [repository-url]
cd MyWorkSalary
dotnet restore
# Öppna i Visual Studio och kör F5
```
